Parágrafos e Quebras de Linha

Você pode escrever um parágrafo de qualquer jeito, basta colocar tudno em uma par de tags <p> e </p>.

Se precisar quebrar o texto em algum lugar específico
como esse, você pode usar a tag <br>.

Vamos adicionar símbolos especiais:
® = Marca Registrada
© = Marca Copyright
™ = Trade Mark
€ = Moeda Euro
£ = Moeda Libra Inglesa
¥ = Moeda Yene
¢ = Moeda Centavos de Dólar
Δ = Delta
↑ = Seta pra cima

Vamos adicionar algums emojis:
🖖 = Saudação Vulcana
🤓 = Carinha com Óculos
